# Portafolio Personal con Docker y Nginx

Este proyecto configura un servidor Nginx con Docker para alojar tu portafolio web.

## Requisitos previos

- Docker instalado
- Docker Compose instalado

## Instalación

1. Clona o descarga este proyecto en tu servidor
2. Ejecuta el script de configuración:
   ```bash
   chmod +x setup.sh
   ./setup.sh
